# Documentation

This folder contains technical documentation for the SplitMS project.

## Contents

- [Architecture](architecture/README.md): Complete system design, architecture decisions, component interactions, behavioral flows, and data model with 10+ Mermaid diagrams.

## Diagram Categories

All diagram sources are written in Mermaid and stored under [docs/architecture/diagrams](architecture/diagrams):

- **Structural Views:** System context, runtime layers, component map, deployment topology
- **Behavioral Views:** Authentication flow, navigation state machine, content loading sequence, expense creation workflow, settlement sequence
- **Data Views:** Entity-relationship diagram (ERD) from Flyway migrations V1–V9
